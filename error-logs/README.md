# 🐛 Docker Error Logs

This folder is dedicated to tracking and documenting errors, issues, and debugging notes encountered while learning and working with Docker.

> **Purpose:** Maintain a personal knowledge base of problems and their solutions so you can debug faster next time!

---

## 📁 Folder Structure

```
error-logs/
├── README.md                   ← You are here
├── TEMPLATE.md                 ← Copy this for every new issue
├── beginner-errors/            ← Issues faced during beginner level
├── intermediate-errors/        ← Issues faced during intermediate level
└── advanced-errors/            ← Issues faced during advanced level
```

---

## 🚀 How to Use

1. **When you face an error**, copy `TEMPLATE.md` and rename it descriptively:
   - Example: `beginner-errors/2026-05-06-docker-daemon-not-running.md`
2. **Fill in all sections** of the template with as much detail as possible.
3. **Once resolved**, mark it ✅ and document the solution clearly.
